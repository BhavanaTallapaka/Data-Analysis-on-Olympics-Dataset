# Olympics Data Analysis

This project performs **Exploratory Data Analysis (EDA)** on historical Olympic Games data to understand patterns in athlete participation, medal distribution, and country performance.

The analysis uses Python data science libraries to explore trends across Olympic events, athletes, and nations.

---

## Project Overview

The Olympic Games generate a large amount of data about athletes, countries, and sports.  
This project analyzes historical Olympic data to identify patterns and insights related to participation, medal distribution, and athlete demographics.

The dataset contains records of athletes competing in Olympic events across multiple years, including details such as country, sport, event, and medals won. :contentReference[oaicite:0]{index=0}

---

## Dataset

The dataset used for this analysis contains information about Olympic athletes and events.

Typical features include:

- Athlete Name
- Gender
- Age
- Height and Weight
- Team / Country
- Year of Olympics
- Sport and Event
- Medal Won (Gold, Silver, Bronze)

Historical Olympic datasets commonly include records from **1896 to modern Olympic Games**, containing thousands of athlete records across many sports. :contentReference[oaicite:1]{index=1}
